# Radio Amateur's Handbook

A reference website about the basics of radio engineering and electronic components. It contains descriptions of resistors, capacitors, transistors, diodes, and inductors, as well as helpful calculators.

## What will you find on the site?

- **Resistors** – how to determine resistance using color bands, plus calculation of total resistance for series or parallel combinations of several resistors.
- **Capacitors** – description of different types, their marking, and a built‑in LC calculator for the resonant frequency of a tank circuit.
- **Transistors** – brief information about bipolar and field‑effect transistors, their main parameters.
- **Diodes** – types of diodes, volt‑ampere characteristic, key specifications.
- **Inductors** – general information and a color‑code calculator for chokes.

## How to use the calculators

### 1. Resistor color code calculator
- Choose the number of color bands.
- For each band, select the color from the drop‑down list.
- The result  appears automatically.

### 2. Total resistance calculator
- Enter the resistance values.  
  Leave unused fields as zero or empty.
- Choose the connection type: series or parallel.
- Click `Calculate total resistance`.

### 3. LC calculator (tank circuit frequency)
- Enter the numeric value of inductance and capacitance.
- Select the units (`µH/mH/H` for L, `pF/nF/µF/mF/F` for C).
- The frequency in Hertz updates automatically.

### 4. Color code calculator for inductors
- Choose 4 or 5 bands.
- Specify the band colors according to the marking on the choke body.
- The result – inductance in µH, mH, or H and tolerance.
